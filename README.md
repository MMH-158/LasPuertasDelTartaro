## Descripción

Simulador de misión heróica ambientado en la mitología griega.
El jugador es un descendiente de un dios seleccionado y debe adentrarse en las profundidades del Inframundo para encontrar tres objetos legendarios y detener el ritual de aperturra de las puertas del Tártaro e impedir que los Titanes escapen. Deberá darse prisa si quiere completar el objetivo, ya que hay mucho terreno desconocido por explorar y el tiempo es limitado.
La ruta se representará gráficamente en Turtle durante la expedición para facilitar la visualización del juego, aunque este se controla exclusivamente por Terminal. 
 
## Controles generales

1. Moverse hacia adelante
2. Cambiar ángulo (para cambiar la dirección de avanze de nuestro personaje)
3. Descansar
4. Mostrar inventario
5. Mostrar zonas del inframundo
